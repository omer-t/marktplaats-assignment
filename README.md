# Marktplaats interview assignments

This repository contains my work for the Marktplaats interview assignments.

There are two main parts:

- `assignments/cars/` contains the notebooks, SQL, helper modules, and source data for the cars analytics assignment.
- `assignments/smb_bundles/` contains the notebooks, helper modules, and source data for the SMB bundles assignment.

The easiest way to review the work is through `deliverables/`:

- `deliverables/cars/` has PDF and HTML exports for each cars question.
- `deliverables/smb_bundles/` has the final SMB bundles PDF, PowerPoint deck, and scored seller output.

The notebooks in `assignments/` are included for transparency and reproducibility. They are intentionally kept fairly code-thin for readability, with most reusable logic moved into the local `modules/` folders. The exported files in `deliverables/` are the intended review copies.
